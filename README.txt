***** Running LIT tests *****
enum34 is required to generate tests: https://pypi.python.org/pypi/enum34#downloads
Clang.exe should be set in PATH variable
cmake .. -G "Visual Studio 12"  -DLIT_EXECUTABLE:FILEPATH=<PATH>\llvm-lit.py
