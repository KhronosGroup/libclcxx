Detailed Description

template<typename T>
struct std::add_generic_pointer_address_space< T >

Provides the member typedef type which is the same as T, except that the address space of the type pointed to by a pointer T is replaced with __generic.

If T is not a pointer type, then type is the same as the T provided.

Template Parameters
T- a pointer type with the address space it points to being replaced by __generic.
